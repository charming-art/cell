# Typography

- `text(text, x, y)`
- `text_width(text)`
- `text_align(align_x=LEFT | RIGHT | CENTER[, align_y=TOP | BOTTOM | MIDDLE])`
- `text_size(size=NORMAL | BIG |LARGE)`
- `text_height(text)`
- `get_font_list()`
- `text_font(name)`
