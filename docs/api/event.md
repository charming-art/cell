# Events

## Keyboard

- `get_key()`
- `get_key_code()`
- `get_key_pressed()`
- `key_pressed(f)`
- `key_typed(f)`
- `key_released(f)`
  
## Mouse

- `get_mouseX()`
- `get_mouseY()`
- `get_mouse_pressed()`
- `get_mouse_button()`
- `mouse_clickd(f)`
- `mouse_pressed(f)`
- `mouse_released(f)`

## Cursor

- `get_cursorX()`
- `get_cursorY()`
- `get_pcursorX()`
- `get_pcursorY()`
- `cursor_moved(f)`
- `cursor_pressed(f)`
- `get_cursor_moved()`
